Git & GitHub Cheetsheet

List of commands

Commands Description
git init Initializes a folder with git
git add . Adds all the modified or newly created files to stage
git commit -m "commit statement" Commit the stage changes to current branch
git status Check the status
git log To see all the commits
git remote add origin URL Connecting Git & GitHub - Local to Empty Remote Repo
git push origin master To push local data to remote
git fetch origin Pull all branches from GitHub
git checkout #GIT ID# To get a specific commit back
git revert #GIT ID# To revert the changes
git reset --hard #GIT ID# To reset the commit
